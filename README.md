# Pathwise AI

Interactive front-end MVP for an AI career copilot.

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.

## What it demonstrates

- Résumé-to-role fit analysis and skill gaps
- Four-week learning roadmap with progress states
- Mock interview answer feedback

The prototype uses local, deterministic logic so it can run with no account or API key. Production AI analysis should run server-side with an `OPENAI_API_KEY`; never expose it in browser code.
